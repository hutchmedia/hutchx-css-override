Manage custom site-wide CSS, header scripts, external-link behaviour, login logo, and receive GitHub-powered updates (public repos).

== Description ==
- Custom CSS injected late in <head>
- Header scripts printed into <head>
- External links toggle adds target="_blank" + rel="noopener noreferrer"
- Custom logo on wp-login
- Lightweight GitHub updater: set repo as `owner/repo` and publish Releases

== Installation ==
1. Upload the ZIP in Plugins → Add New → Upload Plugin.
2. Activate.
3. Go to “HutchX CSS” in the admin menu.
4. (Optional) Under “GitHub Updates” enter your public repo as `owner/repo`.
